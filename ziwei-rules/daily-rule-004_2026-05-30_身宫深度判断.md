## 核心内容

### 一、身宫的本质：命宫是「总统府」，身宫是「行政院」

周信轸老师给出了最精辟的类比：

> **命宫** = 先天格局、整体形象，如一个国家之总统府——代表你的「天命」。
> **身宫** = 后天行为、行动力、价值实现，如行政院/立法院——代表你「怎么做」。

命宫显于**表**（性格底色），身宫显于**行**（实际行为）。一张命盘里，先天格与后天行可能一致，也可能矛盾。当身宫与命宫星曜属性冲突时，此人「说的」和「做的」是两回事——这是解盘必须洞察的关键。

### 二、身宫推算法

> 从寅宫起正月，顺数至生月所在宫位 → 从此宫逆数至生时，即是身宫。

**严苛约束**：身宫永远不可能落入父母、田宅、交友、疾厄、子女、兄弟这六个宫位。若排出的命盘身宫不在命/夫/财/迁/官/福六宫范围内，排盘必然有误。

### 三、身宫落六宫的含义——精力投放的方向标

| 身宫落宫 | 含义 |
|----------|------|
| 命宫 | 生活比较安定，不太愿意变动 |
| 夫妻宫 | 一生最大志向是疼爱配偶、把家顾好 |
| 财帛宫 | 一生最大志愿是好好赚钱 |
| 迁移宫 | 人常常往外跑，心理状态也倾向于在外 |
| 事业宫 | 一生最大希望在事业上有所表现 |
| 福德宫 | 比较趋于享受 |

### 四、深度判断三技法

**① 身宫立太极——「后天六亲」的判断**
身宫不仅影响命主自身行为，还可以「立太极」来论后天行为中的六亲关系。例如：身宫在迁移宫 → 身宫的夫妻宫 = 本命财帛宫 → 判断后天行为中对配偶的态度。

**② 命宫 + 身宫 + 疾厄宫 三宫合看外貌**
优先级：先看命宫 → 次看身宫 → 再参考疾厄宫（一六共宗，疾厄宫是「另一个命宫」）。

**③ 空劫临命身——特殊凶格**
天梁空和地劫分别落在命宫与身宫时，形成「空劫临命身」结构，主一生颠沛、内心空虚、想得得不到（空）、得到了又被咬掉（劫）。

## 关键操作

1. **排盘验证**：身宫是否在命/夫/财/迁/官/福六宫之一 → 不在则排盘有误
2. **解盘时命身合参**：先读命宫（先天性格），再读身宫（后天行为），看两者是否一致
3. **身宫立太极**：以身为太极点，重组十二宫对应关系
4. **三宫合看**：命+身+疾厄 → 综合判断外貌气质与深层人格

## 注意事项

1. 身宫管「行为模式与精力投放」，但大限/流年的运势变化不单由身宫决定
2. 身命冲突是信号不是灾难：先天格与后天行矛盾时，看四化飞宫判断张力是转化为动力还是内耗
3. 空劫在命身不可轻断：若三方有强吉星会照，颠沛中仍有贵人提携
4. 身宫落夫妻≠婚姻一定好：只能说当事人愿意为婚姻付出，婚姻质量还须看夫妻宫本身的星曜和四化
