# 每日规则提取进度

> 目标：每天从课程转录中提取一条斗数规则，结构化输出，日积月累

> 启动日期：2026-05-28

## 提取方法论

从 `ziwei_technique_catalog_v1.md` 的 75 条技法中，优先按以下顺序：
1. **命格主轴**（四化人格、过犹不及、星有本宫）→ 命盘分析第0步核心
2. **体用飞宫**（有效飞宫判断、一五九法、化出化入）→ 第3-4步核心
3. **自化平方**（平方象意、阴阳消长、忌冲）→ 第5步核心
4. **定盘/大限**（命签纸田、流月流日推算法）→ 第0/7步核心
5. **独立模块**（反断+共盘、星宫配合论+财官合一）→ 许铨仁体系

## 规则输出格式

每条规则输出：
- **规则名称**：简短标签
- **来源**：三位老师中的哪位 + 具体课程编号
- **核心内容**：2-5段，清晰解释规则
- **关键操作**：如果涉及操作步骤，列出清单
- **注意事项**：容易踩坑的点
