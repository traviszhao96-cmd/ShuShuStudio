# 左辅右弼星规则

## 来源
钦天门四化基础课程 · 左辅右弼专题

---

## 一、左辅星

### 核心规则
- **左辅属于壬天干**：看左辅在什么宫位，再看壬天干在什么宫位，两个宫位一起解释
- 左辅在什么宫位，就必定存在此宫位的事情，在此人人生里不一般
- 左辅是**实在的星**，务实，眼见为实，"拿到结果"

### 操作步骤
1. 定位左辅所在宫位 → 此宫位有"必定发生"的大事
2. 定位壬天干所在宫位 → 左辅能量的来源
3. 两天宫位同参解释

### 实例速查
| 左辅位 | 壬天干位 | 解读 |
|--------|----------|------|
| 福德 | 交友 | 自己有福气，朋友也因此得福 |
| 福德 | 疾厄 | 自己有福气，但长辈身体有问题 |
| 迁移 | 任意 | 外出在此人生命里重要，会去外地/外国寻机缘 |
| 夫妻 | 田宅 | 有了另一半才形成买房目标 |
| 迁移 | 迁移(同宫) | 左辅离心自化，注意交通事故 |

### 离心自化
- 左辅离心自化 = 能量离开盘面 → 注意特殊年份的损失
- 左辅生年科 + 自化离心科 = 某个物质存在一定会离开
- 左辅在子宫（地支子）时例外 → 有"锁"的作用，反而稳定

### 心性
- 思想上的务实 — 看不到就不相信
- 占据时间，难以放下 → 执念、坚韧、不达目的不罢休
- **无明**：预设立场，看不到真相，容易耿耿于怀
- 喜欢**动态**

### 优化
- 在左辅所在宫位，多听别人想法再决定行为
- 面对弱于自己的不要生太多意见
- "做客不做主" — 不做选择不做主，反而更好

## 二、右弼星

### 核心规则
- **右弼属于戊天干**：先看右弼在什么宫位，再看戊天干在什么宫位，同参
- 右弼是**灵魂的星** — 有先天良意，总有天意的感觉
- 右弼 + 文昌 = 灵魂（右弼为灵，文昌为魂）

### 天干加成
右弼在以下天干时加强先天良意：**丙、戊、己、辛、壬**

### 操作步骤
1. 定位右弼所在宫位 → 此宫位有先天良意
2. 定位戊天干所在宫位 → 右弼能量的演化
3. 通过戊天干看设象星（四化）到何宫位
4. 两天宫位同参

### 实例速查
| 右弼位 | 戊天干位 | 解读 |
|--------|----------|------|
| 子女 | 田宅 | 事业有天命 + 固定年份买房且生子 |
| 官禄 | 迁移 | 工作对外，适合外贸 |
| 夫妻 | 田宅 | 男：自己买房；女：丈夫买房 |
| 命宫 | 迁移（设象科入命） | 女：丈夫给房产；男：因另一半置产 |

### 心性
- 思想受境遇影响，行为受思想所控
- 潜意识 = 先天良执念 = 信仰或固定价值观
- 喜欢**静态**
- 右弼+昌曲左辅同度 → 一辈子面临不止一次大转折

## 三、左右同参

### 核心规则
- **左辅和右弼必须同参**（同宫或产生联系时），否则容易"说反话"
- 左右同宫/有联系 → 拧巴心态，既要又要，患得患失
- 动大于静：左辅的动态 > 右弼的静态

### 能量串联规则
- 左辅 → 查壬天干 → 定位
- 右弼 → 查戊天干 → 定位
- 左右串联时，静态可转化为动态（如安静工作→经常外出）

### 形象比喻
左右昌曲 = 电脑的硬件（主板、内存、显卡）
紫微斗数 = 完整的电脑系统
要了解系统，必须先了解硬件

## 四、赵的命盘应用

赵：左辅在田宅宫，右弼在疾厄宫（左右分隔，未同宫）

### 左辅线：田宅(癸巳) + 壬天干查找
- 赵盘壬天干在**福德宫**
- 左辅在田宅 → 房产/家庭在人生中不一般
- 壬天干在福德 → 能量源于内心追求
- 🔑 同参：因内心追求（福德）而产生置业/家庭（田宅）的实质行动

### 右弼线：疾厄(丁酉) + 戊天干查找
- 赵盘戊天干在**财帛宫**
- 右弼在疾厄 → 身体有先天良意，健康有"天意"感
- 戊天干在财帛 → 能量连到金钱
- 🔑 同参：身体健康与财富之间有深层联系，赚钱方式影响健康状态

### 左右串联
- 左辅(田宅) ←壬→ 福德
- 右弼(疾厄) ←戊→ 财帛
- 田宅+疾厄+财帛+福德 = 物质生活四宫串联
- 左右未同宫，但通过壬戊天干产生联系 → 在安稳(右弼静)和行动(左辅动)之间切换
