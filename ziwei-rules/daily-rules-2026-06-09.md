# 每日斗数规则提取 — 2026-06-09

## 规则 #14：流月流日推算法

**来源**：陈小飞 高阶P14 + 高阶P15（周信轸P17不相关，实际讲跛足马格局）

### 核心发现
- 流月推算完全以本命盘**寅宫（斗君）**的宫位名为锚点
- 公式：本命寅宫名 → 流年盘同名宫 = 正月 → 顺时针排2-12月
- 流日公式：目标流月宫的下一个顺时针宫位 = 初一 → 逐宫顺排
- 生肖位法快速定位：生肖位 = 虚岁1/13/25/37/49/61...
- 推运解读只看宫位重叠（流年命宫叠本命某宫+叠大限某宫），不关注流年层面宫位名
- 解释顺序：生年四化 → 自化 → 飞宫（须遇生年）
- 闰月以十五号为分界
